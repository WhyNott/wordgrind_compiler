BFOI = before first oracle item
AFOI = at first oracle item
BFT = before first token
AFT = at first token


| Kind                | O. Start | O. End | T. Start | T. End |
|---------------------+----------+--------+----------+--------|
| Sentence (oracle)   | BFOI     | ALOI   | AFT      | ALT    |
| Sentence (parse)    | BFOI     | ALOI   | AFT      | ALT    |
| Items (oracle)      | BFOI     | ALOI   | AFT      | ALT    |
| Items (parse)       | BFOI     | ALOI   | AFT      | ALT    |
| Initial (oracle)    | BFOI     | ALOI   | AFT      | ALT    |
| Parameters (oracle) | BFOI     | ALOI   | AFT      | ALT    |
| Elements (oracle)   | BFOI     | ALOI   | AFT      | ALT    |
